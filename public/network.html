What happens?

Segmentation makes the allowed path explicit

Public users should enter through the web app. The API should be the narrow service path to the database. A firewall and internal-only network reduce accidental exposure, while TLS protects the approved traffic path.

Best practice takeaway

Expose the application, not the database.
