      Why does this matter?
      

        
    
          
  • A public database port exposes the service directly to attackers.
    • 
          
  • Network segmentation restricts access paths and reduces attack surface.
    • 
          
  • Firewalls enforce policy before traffic reaches the database.
    • 
        

      

    


    

      

        

          
Security takeaway

          
Network segmentation removes direct database exposure.

          
Expose the application, not the database.

        

        

          
Operational takeaway

          
Keep allowed paths explicit and testable.

          
Connection tests should show which routes are protected, blocked, or exposed.
